Quick Start

1

Download

Download JMC Launcher.exe from the releases page.

2

Place & Run

Place it anywhere; the launcher will auto-create logs and config files in the same folder.

3

Configure

Enter your username in the Launcher tab. Tweak RAM and Java settings in the Settings tab if needed.

Important Notes

  • Windows Only
  • Local storage for logs and config
  • Works offline for launching
  • Internet required for Fabric updates
  • Follows Windows Light/Dark theme
  • Daily logs saved in logs/ folder
